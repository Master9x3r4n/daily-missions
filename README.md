## READ ME ##
How to run:
1. Install Python.
2. Clone the repository
3. Run the following command: ```pip install pygame``` if you haven't got pygame installed.
4. Run ```python main.py```

How to use:
- Click on the __yellow__ button on the bottom right of a card to claim the mission.
- Click on the `R` button on the bottom of the screen to refresh all missions.
- Click on the `+` button to open the Edit Missions on the command screen. Navigate this to add or delete missions to your board.
- Click on the arrow buttons on the bottom of the screen `<` and `>` to navigate the pages.
- You can also edit the data.txt file directly to add new missions, just strictly follow the following format:
```
Mission Name (16 characters max)
Mission Description (200 characters max)
True/False (if you completed the mission)
```
- __NOTE:__ If you do not properly follow the format, the program may crash.
